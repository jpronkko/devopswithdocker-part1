# The dockerized project is an imaginary art gallery web page with baked in assets and served via nginx. The image url is: https://hub.docker.com/u/sirmmidockr . One can use the image for example by a docker run -command and then browsing using the port (here 8080):

docker run -d -p 8080:80 sirmmidockr/nginx-page:latest

# The project is also currently running in render: https://nginx-gallery-page.onrender.com/ (May take a while to open)
